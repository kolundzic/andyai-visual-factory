# Visual Factory Workflow

## Canonical Workflow

```text
Brand Input
→ Agent Planning
→ Variation Grid
→ Human Selection
→ Final Regeneration
→ Export Pack
→ Memory Save
```

## Variation Grid Prompt

```text
Create a 5x5 grid of 25 distinct visual directions.
Number each cell from 1 to 25 in a small circle.
Keep all designs within the same brand system.
Make every cell visually different enough for selection.
```

## Final Regeneration Prompt

```text
Regenerate variant [NUMBER] as a standalone high-resolution final image.
Remove the grid number.
Preserve the main composition.
Improve typography, spacing, contrast, and visual hierarchy.
Keep the text clean and readable.
```

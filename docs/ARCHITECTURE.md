```text
01 Brand Context Layer
   ↓
02 Prompt Planning Layer
   ↓
03 Image Generation Layer
   ↓
04 Variation Review Layer
   ↓
05 Human Selection Layer
   ↓
06 Final Regeneration Layer
   ↓
07 Export Pack Layer
   ↓
08 Memory Save Layer
```

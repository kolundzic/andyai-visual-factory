# Relationship to AndyAI Visual Canon

AndyAI Visual Factory is the production layer built on top of AndyAI Visual Canon.

```text
andyai-visual-canon
        ↓
andyai-visual-factory
```

## AndyAI Visual Canon

Defines:

- visual language
- layout standards
- diagram standards
- icon usage
- typography principles
- color doctrine
- documentation visual rules
- aesthetic consistency

## AndyAI Visual Factory

Produces:

- social visuals
- UI mockups
- infographics
- hero images
- pitch deck graphics
- merch mockups
- visual variations
- export packs

## Canonical Formula

```text
Visual Canon defines.
Visual Factory produces.
```

# Runtime Overview

AndyAI Visual Factory moves from a documented visual canon toward a SaaS-ready visual production runtime.

## Runtime object flow

```text
Production Request
→ Visual Job
→ Job Lifecycle
→ Generation Step
→ Human Review
→ Selection Ledger
→ Export Pack
→ Asset Registry
→ Visual Memory
```

## Core principle

```text
A visual request must become a trackable production object.
```

# Workspace Folder Contract

## Required folders

```text
brand-pack/
jobs/
prompts/
generated/
selected/
exports/
memory/
reports/
```

## Folder responsibilities

| Folder | Purpose |
|---|---|
| brand-pack | brand identity and reusable rules |
| jobs | production job files |
| prompts | compiled prompts and brief outputs |
| generated | model-generated variants |
| selected | human-selected winners |
| exports | final output packs |
| memory | reusable style decisions |
| reports | summaries and evidence |

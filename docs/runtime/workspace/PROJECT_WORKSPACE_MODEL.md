# Project Workspace Model

A Project Workspace is the production room for one brand, client, project, or campaign.

## Purpose

A workspace organizes:

- brand pack
- visual jobs
- prompts
- generated variants
- selected winners
- export packs
- visual memory
- production reports

## Canonical workspace structure

```text
workspace/
├─ brand-pack/
├─ jobs/
├─ prompts/
├─ generated/
├─ selected/
├─ exports/
├─ memory/
└─ reports/
```

## Principle

```text
A job without a workspace is loose work.
A job inside a workspace becomes production.
```

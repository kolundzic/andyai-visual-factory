# Job Status Table

| Status | Meaning |
|---|---|
| draft | Request exists but is not ready |
| planned | Prompt/brief plan exists |
| queued | Waiting for generation |
| generating | Model generation is running |
| review_required | Human needs to review variants |
| selected | Human selected a winning variant |
| regenerating | Final regeneration is running |
| exporting | Export pack is being prepared |
| completed | Job completed successfully |
| archived | Job is stored for reference |
| failed | Job failed and needs attention |
